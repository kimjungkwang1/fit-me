# 🛒🛍️Fit Me

> AI 피팅/추천 패션 쇼핑몰

**피팅 AI**를 통해 사용자가 제품을 미리 입어보고 **사회적 시간과 비용을 절약**

<br />

#  **📘** 프로젝트 진행 기간

2024.02 ~ 202.04 (6주)

<br />


# 👕 주요 기능

* 피팅 AI로 미리 입어보는 **드레스룸 기능**
* 메타 데이터를 활용한 **유사 상품 추천**(Content-based Filtering)
* Batch를 활용한 **실시간 랭킹 / 인기도 계산**

<br />
